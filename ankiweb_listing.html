AI Field Filler — AI-powered auto-fill for Anki note fields AI Field Filler uses LLM-based AI to intelligently auto-fill blank note fields. It considers already-filled fields, per-field instructions you configure, and optional user prompts to generate appropriate content — including text, audio (TTS), and images.

Features

  • Multi-provider support — OpenAI, Anthropic (Claude), Google (Gemini / Nano Banana), and any OpenAI-compatible API
  • Smart field filling — The AI decides what content type fits each field and leaves irrelevant fields empty
  • Text-to-speech — Generates audio files for audio-type fields (OpenAI TTS or Google Gemini TTS)
  • Image generation — Generates images for image-type fields (OpenAI DALL-E or Google Nano Banana)
  • Optional inline images — For text fields, the AI can include a generated illustration when it would help the learner
  • Per-field instructions — Configure what each field should contain per note type
  • Dynamic model selection — Model dropdowns fetch available models from the API with one click
  • Multiple activation methods — Toolbar buttons, right-click context menu, or keyboard shortcuts

Quick Start

  1. Open Tools → AI Field Filler → Settings...
  2. Enter your API key for your preferred provider
  3. Click Test Connection to verify
  4. Go to the Note Types tab and describe what each field should contain
  5. Open a note in the editor and click AI Fill All or press Ctrl+Shift+G

How It Works

  1. When you trigger a fill action, the addon gathers all field values, per-field instructions, and your optional prompt
  2. This context is sent to your AI provider as a structured prompt
  3. The AI returns content for each field, including type decisions (text, audio, or image)
  4. Text fields: content is inserted as HTML. The AI may optionally add an illustration.
  5. Audio fields: text is sent to the TTS provider and saved as a sound file
  6. Image fields: a prompt is sent to the image provider and saved as an image
  7. Fields the AI deems irrelevant are left empty

Usage

  • Fill All Blank Fields: toolbar button, right-click menu, or Ctrl+Shift+G
  • Fill Current Field: toolbar button, right-click menu, or Ctrl+Shift+F
  • Quick Configure: right-click any field → "AI: Configure instructions..." to set up instructions inline

Requirements

  • Anki 2.1.50 or later (Qt6 recommended)
  • An API key for at least one supported provider
  • Internet connection
